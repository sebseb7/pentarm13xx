# Pentarm13xx

small dev-board using an NXP LPC1343


* 72Mhz Cortex-M3 Core
* 32kb flash
* 8kb RAM
* USB (CDC,MSC,HID)
* UART
* SPI
* I2C
* 3 16Bit PWM pins
* 4 32Bit PWM pins
* 3 ADC pins
* 18 GPIO pins
* Powered from USB or 3.3V supply
* 2 LEDs
* 2 Buttons

look for pentstm32f4 if you need a more powerful dev board
